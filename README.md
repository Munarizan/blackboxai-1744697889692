# Sekolah Agama Rendah Rakyat Solathiah

## Project Overview
This project is an interactive web application for Sekolah Agama Rendah Rakyat Solathiah. It provides essential information about the school including organizational charts, class information, subject lists, and teaching schedules. The application is designed to be visually appealing and user-friendly, making use of modern web technologies.

## Installation
To run the project locally, you need to clone the repository and open the `index.html` file in your web browser. Here’s how you can do it:

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate into the project directory:
   ```bash
   cd <project-directory>
   ```
3. Open `index.html` in your preferred web browser.

No additional installations are required since it is a static HTML page utilizing external CSS and JavaScript.

## Usage
Once the application is open in your browser, you'll be presented with the main interface. Key features include:

- **Current Date and Time**: Automatically updates every second.
- **Organizational Chart**: Displays the hierarchy of staff members.
- **Class Information**: Lists all classes with their respective student counts.
- **Subject List**: Shows the subjects taught at the school.
- **Teaching Schedule**: Provides a timetable for daily sessions.
- **Relief Teacher Form**: Allows staff to submit information regarding teacher substitutions.

Fill in the provided form to send notifications about relief teachers.

## Features
- Responsive design compatible with various devices.
- Touch-friendly and intuitive user interface.
- Beautifully styled using Tailwind CSS for modern aesthetics.
- Real-time updates of the current date and time.
- Dynamic submission form for relief teacher management.

## Dependencies
The project relies on the following external libraries:
- **Tailwind CSS** - For styling: [Tailwind CSS](https://tailwindcss.com/)
- **Font Awesome** - For icons: [Font Awesome](https://fontawesome.com/)
- **Google Fonts** - For typography: [Google Fonts](https://fonts.google.com/)

Note: These libraries are included via CDN links in the `index.html` file, and no prerequisites are required.

## Project Structure
The project is structured as follows:
```

```

The project consists mainly of an HTML file that integrates CSS and JS from CDNs. No complex back-end or directory structure is employed, making deployment and usage straightforward.
